Output Feedback

Prev: State Feedback Chapter 8 - Output Feedback Next: Transfer Functions

In this chapter we show how to use output feedback to modify the dynamics of the system through the use of observers. We introduce the concept of observability and show that if a system is observable, it is possible to recover the state from measurements of the inputs and outputs to the system. We then show how to design a controller with feedback from the observer state. A general controller with two degrees of freedom is obtained by adding feedforward. We illustrate by outlining a controller for a nonlinear system that also employs gain scheduling.

  1. A linear system with dynamics

    is said to be observable if we can determine the state of the system through measurements of the input u ( t ) {\displaystyle u(t)} and the output y ( t ) {\displaystyle y(t)} over a time interval [ 0 , T ] {\displaystyle [0,T]} .

  2. The observability matrix for a linear system is given by

    W o = [ C C A ⋮ C A n − 1 ] . {\displaystyle W_{o}={\begin{bmatrix}C\\CA\\\vdots \\CA^{n-1}\end{bmatrix}}.}

    A linear system is observable if and only if the observability matrix W o {\displaystyle W_{o}} is full rank. Systems that are not reachable have "hidden" states that cannot be determined by looking at the inputs and outputs.

    d z d t = [ − a 1 1 0 ⋯ 0 − a 2 0 1 0 ⋮ − a n − 1 0 0 1 − a n 0 0 0 ] z + [ b 1 b 2 ⋮ b n − 1 b n ] u y = [ 1 0 0 ⋯ 0 ] z + D u . {\displaystyle {\begin{aligned}{\frac {dz}{dt}}&={\begin{bmatrix}-a_{1}&1&0&\cdots &0\\-a_{2}&0&1&&0\\\vdots \\-a_{n-1}&0&0&&1\\-a_{n}&0&0&&0\\\end{bmatrix}}z+{\begin{bmatrix}b_{1}\\b_{2}\\\vdots \\b_{n-1}\\b_{n}\end{bmatrix}}u\\y&={\begin{bmatrix}1&0&0\cdots &0\end{bmatrix}}z+Du.\end{aligned}}}

    is said to be in observable canonical form. A system in this form is always observable and has a characteristic polynomial given by

    det ( s I − A ) = s n + a 1 s n − 1 + ⋯ + a n − 1 s + a n , {\displaystyle \det(sI-A)=s^{n}+a_{1}s^{n-1}+\cdots +a_{n-1}s+a_{n},}

    An observable linear system can be transformed into observable canonical form through the use of a coordinate transformation z = T x {\displaystyle z=Tx} .

  4. An observer is a dynamical system that estimates the state of another system through measurement of inputs and outputs. For a linear system, the observer given by

    d x ^ d t = A x ^ + B u + L ( y − C x ^ ) {\displaystyle {\frac {d{\hat {x}}}{dt}}=A{\hat {x}}+Bu+L(y-C{\hat {x}})}

    generates an estimate of the state that converges to the actual state if A − L C {\displaystyle A-LC} is has eigenvalues with negative real part. If a system is observable, then there exists a an observer gain L {\displaystyle L} such that the observer error is governed by a linear differential equation with an arbitrary characteristic polynomial. Hence the eigenvalues of the error dynamics for an observable linear system can be placed arbitrarily through the use of an appropriate observer gain.

  5. A state feedback controller and linear observer can be combined to form a stabilizing controller for a reachable and observable linear system by using the estimate of the state in the feedback control law. The resulting controller is given by

    d x ^ d t = A x ^ + B u + L ( y − C x ^ ) u = − K x ^ + K r r {\displaystyle {\begin{aligned}{\frac {d{\hat {x}}}{dt}}&=A{\hat {x}}+Bu+L(y-C{\hat {x}})\\u&=-K{\hat {x}}+K_{r}r\end{aligned}}}
