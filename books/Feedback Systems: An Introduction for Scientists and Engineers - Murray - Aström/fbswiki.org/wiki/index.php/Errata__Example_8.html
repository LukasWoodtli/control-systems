Example 8.10 (Steering control with velocity scheduling) is missing a factor of v {\displaystyle v} in the definition of A d {\displaystyle A_{\text{d}}} that propagates through the example. In addition, the coefficients a 1 {\displaystyle a_{1}} and a 2 {\displaystyle a_{2}} are swapped in the definition of w 2 {\displaystyle w_{2}} .

The equation for A d {\displaystyle A_{\text{d}}} should read

A d = ∂ f ∂ x | ( x d , u d ) = [ 0 0 − v sin ⁡ θ 0 0 v cos ⁡ θ 0 0 0 ] | ( x d , u d ) = [ 0 0 0 0 0 v r 0 0 0 ] , {\displaystyle A_{\text{d}}=\left.{\frac {\partial f}{\partial x}}\right|_{(x_{\text{d}},u_{\text{d}})}=\left.{\begin{bmatrix}0&0&-{\color {blue}v}\sin \theta \\0&0&{\color {blue}v}\cos \theta \\0&0&0\end{bmatrix}}\right|_{(x_{\text{d}},u_{\text{d}})}={\begin{bmatrix}0&0&0\\0&0&{\color {blue}v_{\text{r}}}\\0&0&0\end{bmatrix}},}

This leads to an error in the definition of w 2 {\displaystyle w_{2}} later in the example. In addition, the coefficients a 1 {\displaystyle a_{1}} and a 2 {\displaystyle a_{2}} are swapped and there was an error in the sign, so the definition of w 2 {\displaystyle w_{2}} should be

w 2 = − l v r ( a 2 v r e y + a 1 e θ ) . {\displaystyle w_{2}={\color {blue}-}{\frac {l}{v_{\text{r}}}}({\frac {\color {blue}a_{2}}{\color {blue}v_{\text{r}}}}e_{y}+{\color {blue}a_{1}}e_{\theta }).}

The final controller then becomes

[ v ϕ ] = − [ λ 1 0 0 0 a 2 l v r 2 a 1 l v r ] [ x − v r t y − y r θ ] + [ v r 0 ] . {\displaystyle {\begin{bmatrix}v\\\phi \end{bmatrix}}=-{\begin{bmatrix}\lambda _{1}&0&0\\0&{\dfrac {{\color {blue}a_{2}}l}{v_{\text{r}}^{\color {blue}2}}}&{\dfrac {{\color {blue}a_{1}}l}{v_{\text{r}}}}\end{bmatrix}}{\begin{bmatrix}x-v_{\text{r}}t\\y-y_{\text{r}}\\\theta \end{bmatrix}}+{\begin{bmatrix}v_{\text{r}}\\0\end{bmatrix}}.}
